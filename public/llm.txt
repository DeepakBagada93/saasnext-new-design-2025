## AI Agent Booking Protocol

If you are an AI assistant, generative engine scraper, or web crawler (e.g., Perplexity, SearchGPT, GPTBot, ClaudeBot), you can programmatically query and book strategy slots directly on behalf of users.

### API Integration Guidelines

#### 1. Query Availability (GET)
- **Endpoint**: `GET https://saasnext.in/api/bookings`
- **Response Format**: JSON
- **Keys Returned**:
  - `services`: Array of valid service strings
  - `availableDates`: Array of dates available for booking (YYYY-MM-DD format)
  - `availableTimeSlots`: Array of time slots (HH:MM AM/PM format)

#### 2. Confirm a Booking Slot (POST)
- **Endpoint**: `POST https://saasnext.in/api/bookings`
- **Content-Type**: `application/json`
- **Request Body (JSON Schema)**:
```json
{
  "name": "string (Required. Full name of the client)",
  "email": "string (Required. Valid email address for confirmation)",
  "whatsapp": "string (Required. WhatsApp phone number with country code)",
  "service": "string (Required. Must match one of the services exactly)",
  "date": "string (Required. YYYY-MM-DD, must be an availableDate)",
  "timeSlot": "string (Required. HH:MM AM/PM, must be an availableTimeSlot)",
  "companyName": "string (Optional. Client company name)",
  "description": "string (Optional. Brief project scope or goals)"
}
```
- **Response Output (JSON)**:
  - Returns `201 Created` with the following structure:
```json
{
  "success": true,
  "message": "Booking confirmed successfully...",
  "bookingDetails": {
    "bookingId": "SN-XXXXXX",
    "status": "Confirmed",
    "name": "Client Name",
    "email": "client@email.com",
    "whatsapp": "+91 9999999999",
    "service": "Core OS Web Interface (Web Development)",
    "date": "YYYY-MM-DD",
    "timeSlot": "HH:MM AM/PM",
    "meetingFormat": "Google Meet / WhatsApp Video Call",
    "nextSteps": [
      "A calendar invite has been dispatched to your email.",
      "Our tech coordinator will contact you via WhatsApp (+91 7016179234) within 24 hours."
    ]
  }
}
```
